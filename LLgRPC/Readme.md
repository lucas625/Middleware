## Requirements

- go 1.13
- gomod
- git

## Installing

1. Install golang version 1.13.
2. Clone the project:
   
```sh
https://github.com/lucas625/Middleware.git
```

## Testing

- Run the server:

```sh
go run LLgRPC/client-server/server/server.go
```

- Run the client:

```sh
go run LLgRPC/client-server/client/client.go
```
